## :notebook: Running

### Requeriments

```bash
# Clone this
$ git clone https://github.com/glauberbrack/react-native-input-mask

# Go to your project folder
$ cd react-native-input-mask

# Install all depencies
$ yarn

# Run your project
$ yarn start
```
